# 仿真验收指南

## 自验收流程

当用户未给出具体验收标准时，按以下步骤自行判断仿真是否成功。

### 1. 基本可行性检查

| 检查项 | 通过条件 |
|--------|----------|
| 几何构建 | 无报错，域数 = 预期矩形数 |
| 网格生成 | 无报错，单元质量 > 0.1 |
| 求解收敛 | SolEst/LinErr < 1e-3 |
| 导出成功 | 文件有数据行 |

### 2. 静电场验证

**电压范围：** 施加电压的下限 ≤ V ≤ 施加电压的上限
- 若施加 ±1000V → V∈[-1000, 1000]
- 若施加 0~1000V → V∈[0, 1000]

**电场量级：**
- 平行板电容: E ≈ V/d（允许边缘效应使最大值偏高 20-50%）
- 不符合此规律 → 边界条件可能设错

### 3. 固体力学验证

**位移符号：**
- 静电力应拉向电极方向
- 固定端位移 = 0
- 自由端位移最大

**位移量级：**
- 悬臂梁尖端挠度: δ ≈ F·L³/(3·E·I)
- F ≈ ε₀·V²·A/(2·gap²)（平行板静电力）
- 若仿真与理论差一个数量级以上 → 检查约束和材料

### 4. 常见失败模式

| 现象 | 可能原因 |
|------|----------|
| V 恒为常数 | 没设接地/参考电位 |
| E = 0 处处 | 两个导体同电位 |
| 位移 = 0 | 固定约束选错边界 / 静电耦合力未传递 |
| 不收敛 | 约束不足（刚体位移）/ 网格太粗 / 非线性太强 |
| 位移方向反了 | 电压符号反了 |

### 5. 验收报告格式

```
=== 仿真验收报告 ===
模型: [描述]
V 范围: [min] ~ [max] V （施加: [施加值]）
|E| 最大: [值] V/m （理论: [值]）
位移最大: [值] μm （方向: [描述]）
求解状态: 收敛 / 不收敛
结论: 通过 / 需调整
```
